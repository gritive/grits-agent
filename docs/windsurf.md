# Grits + Windsurf

## Setup

Add to your Windsurf MCP settings:

```json
{
  "mcpServers": {
    "grits": {
      "command": "grits",
      "args": ["mcp"]
    }
  }
}
```

## Prerequisites

1. Install: `brew tap gritive/tap && brew install grits`
2. Authenticate: `grits login`
3. Restart Windsurf after adding MCP configuration
